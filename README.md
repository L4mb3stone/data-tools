# Data-tool

## 介绍 📘

​ Data-tool 是一款基于**vue3**、**arco design**、**electron** 开发的数据分析工具
## 项目预览 👀



## 代码仓库 🌟

- GitHub：https://github.com/FengNu/data-tools

## 快速开始 🚀

- **拉取项目** (您需要先安装 Git)

```bash
# GitHub
git pull https://github.com/FengNu/data-tools.git
```

- **运行项目**


```bash
cd 拉取项目目录
npm install	    # 安装依赖
npm run dev    # 运行vue程序
npm run start	# 运行electron桌面程序
```
